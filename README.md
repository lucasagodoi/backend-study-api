# Backend Study API

Simple REST API built with Node.js and Express to practice CRUD operations.

## 🚀 Technologies
- Node.js
- Express

## 📌 Features
- GET / → API status
- GET /items → List all items
- POST /items → Create new item
- DELETE /items/:id → Remove item

## ▶️ How to run locally

1. Clone the repository
2. Run:
   npm install
3. Start the server:
   npm start

The server will run on port 3000.
